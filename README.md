# Audio Monitor Recorder

マイク入力を録音しながら、リアルタイムでスピーカーから再生できるシンプルな音声録音ソフトです。

## 主な機能

* マイク音声の録音
* 録音中の音声をスピーカーに出力（モニター）
* WAV / MP3形式で保存
* ファイル名の指定
* 保存先フォルダの選択
* 録音時間の表示
* シンプルで直感的なUI

## 使い方

1. マイクを選択
2. 保存先フォルダを選択
3. ファイル名と形式を設定
4. 「録音開始」をクリック
5. 「停止して保存」で録音完了

## 配布について

このソフトは7z形式で配布されています。
ダウンロード後、解凍してEXEファイルを実行してください。

## 注意

* マイクとスピーカーが近いとハウリングが発生する場合があります
* 初回起動時は保存先フォルダを設定してください
* Windowsのセキュリティ警告が表示される場合がありますが、「詳細情報」→「実行」で起動できます

## 動作環境

* Windows 10 / 11
* .NET 6（単体EXE版は不要）
